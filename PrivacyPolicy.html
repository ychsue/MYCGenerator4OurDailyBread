Privacy Policy:

When you want to use this App to create a container with a category folder inside for MemorizeYC, at the first time it will ask you to pick a folder as a bookshelf from your file system to keep all the category folders that it will create. After that, it will create a category folder into your selected folder automatically when you ask it to create one.

By default, the folder for FolderPicker is set as the Documents folder; however, you have better to create a new folder under it to hold all these container folders since it is not a good idea to see a new folder be created in your Documents folder every month.

By the way, all the files inside this bookshelf folder will be text files. You can edit them or even delete them. But if you do so, the contents shown in MemorizeYC will be changed since they are the materials for your study.

Briefly speaking, none of your personal information except your preferred language will be used or stored and this App will send nothing out except the request for Our Daily Bread's contents (If out of this, how can I show those contents?) so that it is out of security problem. What this App doing is just to dump the information shown on this App into that folder just selected by you; those contents are out of your personal information. By the way, the information of your preferred language is just used to determine what language should be selected for your 1st time using this App. Since none of your personal information will be stored or be sent out so that there is out of security problem.

中文

隱私權策略:

當你想要利用這個App產生一個MemorizeYC使用的容器與類別時，第一次做這動作時，它會請你選取一個目錄(就像書櫃一樣)做為盛裝所有這些製造給Our Daily Bread 的容器；以後每次你再次要求它產生一個類別目錄時它就都會自動將類別目錄存到你所選定的目錄裡。

檔案總管的FolderPicker預設是取用你的文件目錄來當該書櫃，不過，我建議你在其下再開一個目錄來盛裝，這樣可以讓你的文件目錄不至於每個月多一個容器目錄出來。

此外，該目錄裡的所有檔案都是文字檔，你可以對他們任意編輯或刪除都沒關係。歡迎編輯他們。不過，一旦修改，則會影響MemorizeYC怎樣顯示他們，因為這些檔案就是你要學習的東西的內容啊！

簡言之，這個APP完全不會用到、不會儲存使用者的個人資訊(除了使用者喜好的語言外，這個只用在一開始決定要顯示哪個語言的內容當內文用而已，若沒有這個，那這APP就不能一開始就將內文切換到比較接近使用者所使用的語言的了)，再者，它除了必須送出要求取得靈命日糧網頁的內文的請求外(不然就看不到內文了，任何瀏覽器都必須做這事，不然網站怎會知道要送甚麼內文給瀏覽器？)它不會送任何的訊息出去，所以完全沒有安全問題。這個APP只將它所顯示的資料存到使用者所選的目錄裡頭罷了，這些訊息裡完全沒有使用者的個人資訊。所以，這個APP既不會也不想使用與儲存使用者的個人資訊，所以，請別擔心你的個人資訊因使用本APP而受到損傷。
